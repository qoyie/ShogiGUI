# ShogiGUI
GUI for chichiwan/99shogi
こっちの環境では99shogiがSegmentation faultが起きたからテスト出来なかった。

# Usage
`java ShogiGUI -h`<br>
`java ShogiGUI [executable file]`<br>
<br>
<br>
`               -h Show this help.`<br>
<br>
`[executable file] Executable file of 99shogi.`<br>
`                  Default: a.out`<br>
